TW Company front-end with Data Vitualization
===========

## Features
- 以公司名稱或董監事搜尋
- 公司投資關係圖
- 同地址公司
- 董監事關係圖

## Run
0. Build python virtual environment
1. pip install -r requirements.txt
2. gunicorn twcomapp:app
